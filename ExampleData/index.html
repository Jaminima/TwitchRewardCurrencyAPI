OwlCoinAPI

A pretty shit WebAPI
Intended to allow OwlCoin to be integrated into almost anything.

Request Examples 

Using cUrl

curl -X GET 'http://54.38.214.166/all/users'
Is the most basic request possible and will return an unordered list of all users present in the database.
This request along with ALL Get requests require no authentication,
Hence they perform no changes or return private data.

curl -X POST 'http://54.38.214.166/user' -H 'DiscordId: 488136134866632704' -d ''
This request contains headers, which hold Key-Value pairs, they only hold single units of data.
So where larger data sets are sent we will have to send them using the data stream.
This command also has no authentication.

curl -X POST 'http://54.38.214.166/account/give/258' -H 'AuthToken: YOURTOKEN' -H 'Value: 0' -d ''
This command contains an AuthToken, which you will be provided if you fill out this form
Without a valid token, the requested action will not be carried out.

curl -X POST 'http://54.38.214.166/update/user' -H 'AuthToken: YOURTOKEN' -d 'UserJson'
In this request, the data stream will contain a formatted json, which will represent the user
we have made changes too.
